- `` `timescale <base> / <precision>`` vor Modul spezifiziert
	- `<base>` spezifiziert welche Basis die Verzögerungen haben
- Verzögerung `# <tval>;` für `<tval>`$\cdot$`<base>`
	- `#1;` bedeutet im Beispiel also eine Verzögerung von 1ns
- Genauigkeit `<precision>`, auf welche die Verzögerungszeit gerundet wird
- Block `<always>` heißt auch Prozess
- Wird endlos wiederholend ausgeführt
- Der Beispielcode synthetisiert jedoch nicht! Aus ihm lässt sich keine gültige Schaltung erstellen
